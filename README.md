# Proyecto_Intermodular_1-DAW

El Proyecto Intermodular de 1º de DAW tiene como eje principal el diseño, desarrollo
y documentación de un portal web completo, pensado para dar solución a una
necesidad real de una empresa simulada o de un contexto profesional creíble.

El objetivo no es solo crear páginas web, sino vivir todo el proceso real que sigue un
desarrollador web profesional:

● Análisis de una necesidad o problema

● Diseño de la arquitectura web

● Maquetación y diseño de interfaces

● Programación frontend y backend (según nivel)

● Gestión de datos

● Documentación del proyecto

● Control de versiones

● Presentación profesional del producto
